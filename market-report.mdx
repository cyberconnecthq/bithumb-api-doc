<h2 style={{ fontSize: "22px", marginBottom: 8 }}>Overview</h2>

<p style={{ fontSize: 15, lineHeight: 1.8 }}>
  The <strong>Market Report</strong> is a consolidated daily Korean overview of the Bithumb-KRW market.
  Rather than a single endpoint, it is a composition over five Market API endpoints — each refreshes on
  the same daily cadence and shares the same publish time.
</p>

<Note>
  Refresh: **daily at 18:00 KST**. Promotion to public read tables completes by **17:50 KST**. The Market
  Report surface is **text + ranked lists only** — chart payloads belong to the AI Report surface.
</Note>

<h3 style={{ marginTop: 24 }}>Common conventions</h3>

<AccordionGroup>
  <Accordion title="Cadence and reference time">
    All Market Report endpoints refresh **daily at 18:00 KST** with the **10-minute pre-publish offset**
    (promotion completes by **17:50 KST**). The QA reference time for sample-data validation is **`07:50 KST`**;
    articles published after `07:50` are excluded from that day's sample.
  </Accordion>

  <Accordion title="Surface scope">
    Market Report is **text + ranked lists only**. Charts are scoped to the AI Report surface only
    (Slack 2026-01-12 Hugh — 🟢). Banded indicators (Fear & Greed) and breadth counts are textual.
  </Accordion>

  <Accordion title="Korean-only output, KST timestamps, KRW currency">
    All user-facing strings are Korean. Timestamps are **Asia/Seoul (KST)**, wire format
    `YYYY-MM-DD HH:MM:SS`. Currency is **KRW (`원`)** with `만 / 억 / 조` magnitude abbreviations and
    thousand-separators on raw numbers. UTC and USD are forbidden in user-facing fields.
  </Accordion>

  <Accordion title="Sentence endings">
    Korean prose uses **음/슴체** (`~함`, `~판단됨`, `~예상됨`). Banned endings: `입니다`, `습니다`,
    `합니다`, `해요`, `있습니다`, `~다.`, `~입니다.`.
  </Accordion>
</AccordionGroup>
