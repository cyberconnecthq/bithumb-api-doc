<div style={{ maxWidth: 760 }}>

<h2 style={{ fontSize: "22px", marginBottom: 8 }}>Overview</h2>

<p style={{ fontSize: 15, lineHeight: 1.8 }}>
  The <strong>Sell Recommendation</strong> API is one of three views over the same hourly AI Recommendation
  batch. It returns assets in the Bithumb-KRW Top-200 universe with <code>total_score &lt; 4.0</code>,
  sorted by score ascending (weakest first), with a hard cap of <strong>50 entries</strong>.
</p>

<Note>
  Neutrals (`4.0`–`5.9`) are excluded. Only assets with `total_score < 4.0` appear on the sell list.
</Note>

<Warning>
  All score fields render with **exactly one decimal place** including trailing zero (`3.0`, `2.5`) — never
  bare integers. The Mintlify viewer may drop the trailing `.0` cosmetically; the actual response values are correct.
</Warning>

<h3 style={{ marginTop: 20 }}>Cadence & freshness</h3>

<CardGroup cols={2}>
  <Card title="Refresh" icon="clock">
    Hourly at KST `:00`. Promotion completes by `:50` of the previous hour.
  </Card>
  <Card title="No padding" icon="filter">
    When fewer than 50 assets fall below `4.0`, fewer items are returned. Neutrals are never used to fill.
  </Card>
</CardGroup>

<h2 style={{ fontSize: "22px", marginBottom: 8 }}>Response Attributes</h2>

<ResponseField name="generated_at" type="string" required>
  Snapshot wall-clock in **Asia/Seoul (KST)**.
</ResponseField>

<ResponseField name="uid" type="string" required>
  Batch identifier. Format: `sell_recommendation_YYYYMMDD_HHMM` (KST), `HHMM` aligned to the hourly cadence.
</ResponseField>

<ResponseField name="assets" type="array of objects" required>
  Up to 50 entries, sorted by `total_score` ascending. Each item:
  <Expandable title="properties">
    <ResponseField name="symbol" type="string" required>
      Bithumb exchange symbol (uppercase).
    </ResponseField>
    <ResponseField name="name" type="string" required>
      Korean asset name.
    </ResponseField>
    <ResponseField name="image_url" type="string" required>
      Asset icon URL.
    </ResponseField>
    <ResponseField name="uid" type="string" required>
      Per-asset document ID. Format: `sell_recommendation_<SYMBOL>_YYYYMMDD_HHMM` (KST).
    </ResponseField>
    <ResponseField name="rank" type="integer" required>
      Position within this view, `1`–`50`. `rank = 1` is the lowest-scoring (weakest) asset.
    </ResponseField>
    <ResponseField name="total_score" type="number" required>
      Overall score `0.0`–`3.9` (filter: `< 4.0`), one decimal place.
    </ResponseField>
    <ResponseField name="score_by_category" type="object" required>
      Object with the six number keys `upside_energy / trading_activity / search_trend / price_stability /
      market_sentiment / price_resilience`. Each `0.0`–`10.0`, one decimal place. Korean labels and
      definitions match the [AI Recommended Assets scoring model](/ai-recommended-assets#the-six-metric-scoring-model).
    </ResponseField>
    <ResponseField name="title" type="string" required>
      Korean headline derived from `summary` body. Max 20 Korean characters.
    </ResponseField>
    <ResponseField name="summary" type="string" required>
      Multi-bullet Korean markdown synthesizing the 6 metric scores plus sentiment trend.
      Each bullet uses `\n- 헤드라인: 본문` form.
    </ResponseField>
  </Expandable>
</ResponseField>
