<div style={{ maxWidth: 760 }}>

<h2 style={{ fontSize: "22px", marginBottom: 8 }}>Overview</h2>

<p style={{ fontSize: 15, lineHeight: 1.8 }}>
  The <strong>Major Events Tonight</strong> API returns a list of upcoming events likely to affect market
  visibility for the day — FOMC, interest-rate decisions, US/Korea earnings, regulatory milestones.
  Limited to **Korean and US events**.
</p>

<Note>
  Powered by the daily Market Report pipeline (refresh **18:00 KST**, promotion completes by **17:50 KST**).
  At least **5 events** by importance level are returned per snapshot.
</Note>

<h3 style={{ marginTop: 20 }}>Query parameters</h3>

<ParamField query="date" type="string">
  Snapshot date in `YYYY-MM-DD` (KST). When omitted, the most recent snapshot is returned.
</ParamField>

<ParamField query="raw" type="boolean" default="false">
  When `true`, returns the **pre-polish** raw model output. Used by Bithumb QA; not for production traffic.
</ParamField>

---

<h2 style={{ fontSize: "22px", marginBottom: 8 }}>Response Attributes</h2>

<ResponseField name="datetime" type="string" required>
  Reference time the snapshot represents in **Asia/Seoul (KST)**. Wire format: `YYYY-MM-DD HH:MM:SS`.
</ResponseField>

<ResponseField name="publish_time" type="string" required>
  Wall-clock when this snapshot was promoted, KST `YYYY-MM-DD HH:MM:SS`.
</ResponseField>

<ResponseField name="major_events" type="array of objects" required>
  At least 5 entries, ordered by event start time. Each item:
  <Expandable title="item properties">
    <ResponseField name="event_name" type="string" required>
      Korean event name. Plain text, no Markdown. Example: `"미국 FOMC 의사록 공개"`,
      `"비트코인 콘퍼런스 개막"`, `"한국은행 5월 금융통화위원회"`.
    </ResponseField>
    <ResponseField name="datetime" type="string" required>
      Event start time in **Asia/Seoul (KST)**. Wire format: `YYYY-MM-DD HH:MM:SS`.
    </ResponseField>
  </Expandable>
</ResponseField>

---

<h2 style={{ fontSize: "22px", marginBottom: 8 }}>Errors</h2>

| Status | Code | When |
|---|---|---|
| <span style={{color:"#d97706",fontWeight:600}}>● 400</span> | `INVALID_PARAMS` | `date` is not `YYYY-MM-DD`. |
| <span style={{color:"#d97706",fontWeight:600}}>● 404</span> | `NOT_FOUND` | No snapshot exists for `date`. |
| <span style={{color:"#dc2626",fontWeight:600}}>● 500</span> | `INTERNAL_ERROR` | Unexpected upstream failure. |
