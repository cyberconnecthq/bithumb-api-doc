---

<div style={{ maxWidth: 760 }}>

<h2 style={{ fontSize: "22px", marginBottom: 8 }}>Overview</h2>

<p style={{ fontSize: 15, lineHeight: 1.8 }}>
  The <strong>Key Trends</strong> API returns Korean trend summaries — macro events, KR-specific drivers,
  and crypto-market narratives — likely to impact the day's market. The wire response is a
  <strong> 2-D array</strong>: an outer array of versions, each version is an array of trend events.
</p>

<Note>
  Powered by the daily Market Report pipeline (refresh **18:00 KST**, promotion completes by **17:50 KST**).
</Note>

---

<h2 style={{ fontSize: "22px", marginBottom: 8 }}>Response Attributes</h2>

<ResponseField name="datetime" type="string" required>
  Reference time the snapshot represents in **Asia/Seoul (KST)**. Wire format: `YYYY-MM-DD HH:MM:SS`.
</ResponseField>

<ResponseField name="publish_time" type="string" required>
  Wall-clock when this snapshot was promoted, KST `YYYY-MM-DD HH:MM:SS`.
</ResponseField>

<ResponseField name="key_trend_events" type="array of arrays of objects" required>
  **2-D structure.** The outer array contains multiple "versions" of the trend lineup; the inner array is
  a sequence of trend events for that version. Bithumb's FE selects one version to render based on user
  preference.

  <Expandable title="event properties (inner item)">
    <ResponseField name="title" type="string" required>
      Korean headline for the trend event, ≤1 line. Plain text, no Markdown.
    </ResponseField>
    <ResponseField name="summary" type="string" required>
      Korean prose summary of the trend's market relevance, ~1–2 sentences. Sentence endings use 음/슴체.
    </ResponseField>
  </Expandable>
</ResponseField>

---

<h2 style={{ fontSize: "22px", marginBottom: 8 }}>Errors</h2>

| Status | Code | When |
|---|---|---|
| <span style={{color:"#d97706",fontWeight:600}}>● 400</span> | `INVALID_PARAMS` | `date` is not `YYYY-MM-DD`. |
| <span style={{color:"#d97706",fontWeight:600}}>● 404</span> | `NOT_FOUND` | No snapshot exists for `date`. |
| <span style={{color:"#dc2626",fontWeight:600}}>● 500</span> | `INTERNAL_ERROR` | Unexpected upstream failure. |

</div>

```bash cURL

  --url "https://api.asksurf.ai/surf-ai/v1/market/trends" \
```

```python Python
import requests

response = requests.get(
    "https://api.asksurf.ai/surf-ai/v1/market/trends",

)
trends = response.json()
```

