<div style={{ maxWidth: 760 }}>

<h2 style={{ fontSize: "22px", marginBottom: 8 }}>Overview</h2>

<p style={{ fontSize: 15, lineHeight: 1.8 }}>
  The <strong>Market Sentiment</strong> API returns a snapshot of overall sentiment signals for the
  Bithumb-KRW market: a Fear &amp; Greed index, breadth counts (Up / Flat / Down assets), and the
  number of assets rising versus the previous day.
</p>

<Note>
  Powered by the daily Market Report pipeline (refresh **18:00 KST**, promotion completes by **17:50 KST**).
  Used by the Bithumb FE as a top-of-page market-pulse indicator.
</Note>

<h3 style={{ marginTop: 20 }}>Query parameters</h3>

<ParamField query="date" type="string">
  Snapshot date in `YYYY-MM-DD` (KST). When omitted, the most recent snapshot is returned.
</ParamField>

<ParamField query="raw" type="boolean" default="false">
  When `true`, returns the **pre-polish** raw model output. Used by Bithumb QA; not for production traffic.
</ParamField>

<h3 style={{ marginTop: 20 }}>Fear & Greed banding</h3>

| `fear_greed_index` | Korean band | English |
|---|---|---|
| 0 – 24 | 극단적 공포 | Extreme Fear |
| 25 – 44 | 공포 | Fear |
| 45 – 55 | 중립 | Neutral |
| 56 – 75 | 탐욕 | Greed |
| 76 – 100 | 극단적 탐욕 | Extreme Greed |

The `fear_greed_interpretation` string is a Korean prose interpretation derived from the index value
(e.g. *"투자 심리가 다소 위축된 상태로, 신중한 접근이 필요한 구간으로 판단됨."*).

<h2 style={{ fontSize: "22px", marginBottom: 8 }}>Response Attributes</h2>

<h3 style={{ marginTop: 24, marginBottom: 8 }}>Snapshot timing</h3>

<ResponseField name="report_time" type="string" required>
  Reference time the snapshot represents in **Asia/Seoul (KST)**. Wire format: `YYYY-MM-DD HH:MM:SS`.
  For sample-data validation runs the canonical reference is `YYYY-MM-DD 07:50`.
</ResponseField>

<ResponseField name="publish_time" type="string" required>
  Wall-clock when this snapshot was promoted to the public read tables (KST, `YYYY-MM-DD HH:MM:SS`).
</ResponseField>

<h3 style={{ marginTop: 24, marginBottom: 8 }}>Sentiment indicators</h3>

<ResponseField name="fear_greed_index" type="integer" required>
  Fear & Greed score. Range: `0`–`100` (higher = more greed).
</ResponseField>

<ResponseField name="fear_greed_interpretation" type="string" required>
  Korean prose interpretation of the index, ~1 sentence. 음/슴체 sentence ending.
</ResponseField>

<h3 style={{ marginTop: 24, marginBottom: 8 }}>Market breadth</h3>

<ResponseField name="rising_assets_vs_previous_day" type="integer" required>
  Count of Bithumb-KRW assets whose 24-hour change is positive compared to the prior day's snapshot.
  Non-negative; bounded by the size of the Bithumb-KRW universe.
</ResponseField>

<ResponseField name="market_price_fluctuations" type="array of objects" required>
  Breadth counts across the entire Bithumb-KRW market — typically 3 entries (Up / Flat / Down).
  <Expandable title="item properties">
    <ResponseField name="type" type="string" required>
      One of:
      | Value | Meaning |
      |---|---|
      | `UP` | Asset's 24-hour change is positive |
      | `FLAT` | Change is zero / within rounding |
      | `DOWN` | Change is negative |
    </ResponseField>
    <ResponseField name="num" type="integer" required>
      Number of assets in this bucket. Sum across all three entries equals the size of the Bithumb-KRW
      market at `report_time`.
    </ResponseField>
  </Expandable>
</ResponseField>

| Status | Code | When |
|---|---|---|
| <span style={{color:"#d97706",fontWeight:600}}>● 400</span> | `INVALID_PARAMS` | `date` is not `YYYY-MM-DD`. |
| <span style={{color:"#d97706",fontWeight:600}}>● 404</span> | `NOT_FOUND` | No snapshot exists for `date`. |
| <span style={{color:"#dc2626",fontWeight:600}}>● 500</span> | `INTERNAL_ERROR` | Unexpected upstream failure. |
